Na aula 3 demos prosseguimento as propriedades CSS e aprendemos como os elementos são posicionados na tela.

A propriedade display

A propriedade display permite que você defina a maneira como determinado elemento HTML deve ser renderizado. Os principais tipos de display são:

  • Block
  • Inline
  • None
  • Inline-block
  • Flex

Display block

A declaração display: block faz com que o elemento HTML seja renderizado como bloco, tal como os parágrafos e os cabeçalhos o são. Um bloco contém um espaço em branco tanto em cima como embaixo e não permite outros elementos HTML ao lado, exceto quando tiver sido declarado ao contrário (por exemplo, declarar a propriedade float para o elemento próximo ao bloco).

Display inline

A propriedade display: inline faz com que o elemento HTML seja renderizado inline, dentro do bloco na mesma linha. Quando o elemento encontra-se entre dois blocos ele forma o chamado 'bloco anônimo' e é renderizado com a menor largura possível.

Display none

A declaração display: none faz com que o elemento HTML não seja renderizado,ou seja, ele não aparecerá na tela.

Display inline-block

Esse display herda comportamento tanto do display block quanto do display inline. Com ele,é possível definir uma width e um heigth, assim como no block e também deixar um elemento do lado do outro.

Display flex

Flex Container

O display flex é bastante utilizado pelos desenvolvedores por ser mais flexível que os demais. Com ele, podemos posicionar os elementos com maior facilidade através de suas propriedades. Quando colocamos a propriedade flex em um elemento, esse elemento se torna um flex container e seus elementos filhos, por padrão, ficam um do lado do outro como se estivessem sob o efeito do display inline.

Propriedades do Flex Container

  1. Flex-direction: indica o eixo principal de um flex-container.É de acordo com ela que o navegador organiza os próximos elementos colocados dentro do flex-container. Por padrão, o eixo principal é o row.
    • row
    • row-reverse
    • column
    • column-reverse
  2. Justify-content: Essa prorpiedade ajusta horizontalmente os elementos filhos do flex container
    • flex-start
    • flex-end
    • center
    • space-between
    • space-around
    • space-evenly
  3. Align-items: Essa propriedade ajusta verticalmente os elementos filhos do flex container
    • stretch
    • flex-start
    • flex-end
    • center
    • baseline
  4. Flex-wrap: Essa propriedade trabalha com a "quebra de linha" dos elementos em linha.
    • nowrap
    • wrap
    • wrap-reverse
