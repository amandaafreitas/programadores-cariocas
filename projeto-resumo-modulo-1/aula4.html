Float

A propriedade float do CSS determina que um elemento deve ser retirado do seu fluxo normal e colocado ao longo do lado direito ou esquerdo do seu containêr, onde textos e elementos em linha irão se posicionar ao seu redor.No exemplo abaixo, os 3 containers fazem uso da propriedade float, para se manterem posicionados a esquerda. Note que o parágrafp lorem não sai do seu fluxo para se posicionar a esquerda. Isso acontece por causa da propriedade clear, que é usava para limpar o fluxo. Caso o contrário, todos os elementos seguintes continuariam se posicionando a esquerda.
