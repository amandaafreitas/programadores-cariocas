Estrutura básica do HTML

A Estrutura básica do HTML se resume :

  1. html: É a raiz do documento, engloba todos os elementos HTML
  2. head: É onde ficam as tags de configuração e o título que aparece na aba do navegador. Configuração de caracter, linguagem.
  3. body: É o corpo do HTML, onde fica o que é visível na tela, o que aparece para o usuário

Tags principais

Algumas das tags mais usadas são:

  • p: tag de parágrafo
  • h Existe o h1, que é o título principal e o h2, h3,h4 que são os subniveis do título principal
  • ul e ol:São listas que podem ser não ordenadas ul e ordenad ol, os itens da lista são os li
