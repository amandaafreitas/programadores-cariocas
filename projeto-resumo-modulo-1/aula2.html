O que é CSS?

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português significa Folha de Estilo em Cascatas. Ele é utilizado para documentos, adicionando cores, margem, formatações, fontes, animações e muitas outras coisas.

Como aplicar CSS

Existem 3 tipos de folha de estilo CSS possíveis para utilização em projetos web:

  1. Inline: É quando os estilos CSS são aplicados diretamente dentro da tag de abertura de um elemento no HTML
  2. Interno:Nesse método, você declara a folha de estilo diretamente dentro da seção head do documento HTML
  3. Externo: Esse é o tipo de uso de CSS mais indicado e recomendado.Nesse tipo,os códigos CSS ficam em um documento externo ao documento HTML,separando assim as camadas de estrutura e apresentação do projeto web.

Seletores

No CSS precisamos saber quais elementos serão estilizados.Para isso precisamos de duas coisas:selecioná-los de alguma forma e aplicar alguma regra

Podemos selecionar elementos de 3 formas principais:

  • Seletor de tag: O seletor de tag é o seletor mais simples do CSS.Para defini-lo basta colocar o nome da tag onde você deseja aplicar a formatação antes da definição das regras de formatação.
  • Seletor de classe: Definimos uma classe na tag de abertura de um elemento HTML.Essa classe pode representar vários elementos, e um elemento pode possuir mais de uma classe. As classes são identificadas com um . (ponto) e possuem a sintaxe .nome-da-classe.
  • Seletor de id: Em uma tag HTML, é possível especificar um identificador, chamado de id. Uma tag pode ter apenas um id, e este id não pode se repetir no resto do documento. Os ids são identificados com um #, então um seletor de CSS de id possuiu a sintaxe #nome-do-id.

Cores

As cores são elementos essenciais na estilização e podemos. Elas podem ser representadas de 3 formas no CSS

  • Nome oficial da cor
  • Sistema rgb()
  • Valor hexadecimal

Box model

Os elementos HTML se comportam como pequenas caixas. Nós podemos alterar algumas propriedades dessas caixas utilizando o CSS. O modelo de caixa possui os seguintes dados.

  • Margem externa: margin
  • Borda: border
  • Margem interna: padding
  • Tamanho: width e heigth
